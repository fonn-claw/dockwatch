**3. Create `src/lib/actions/assets.ts`:**
- "use server" directive
- Import from `zod/v4` (NOT `zod` — per project convention)
- **createAsset(formData: FormData):** requireRole(["manager"]), validate with Zod schema (name string 1-255, assetType enum, dockId number, location string, installDate optional string, warrantyExpiry optional string, conditionRating 1-5 default 3, notes optional), insert into assets table, call logAudit with action="create", entityType="asset", revalidatePath("/assets"), return { success: true, id }
- **updateAsset(id: number, formData: FormData):** requireRole(["manager"]), same Zod schema as create, update where id matches, call logAudit action="update", revalidatePath("/assets"), return { success: true }
- **deactivateAsset(id: number):** requireRole(["manager"]), set deletedAt=new Date() and isActive=false, call logAudit action="deactivate", revalidatePath("/assets"), return { success: true }
- **updateConditionRating(assetId: number, rating: number):** requireRole(["manager", "crew"]), validate rating 1-5, update conditionRating + updatedAt, call logAudit action="update_condition" with metadata { oldRating, newRating }, revalidatePath("/assets"), return { success: true }

**4. Create `src/lib/queries/assets.ts`:**
- Export `getAssets(filters: { dockId?: number; assetType?: string; minCondition?: number })` — query assets with `isNull(assets.deletedAt)` filter, join dock via `with: { dock: true }`, apply optional filters using `and()`, order by `asc(assets.name)`
- Export `getAssetDetail(id: number)` — query single asset with dock relation AND last 5 work orders for that asset (using db.query.assets.findFirst with `with: { dock: true, workOrders: { limit: 5, orderBy: desc, with: { assignee: true } } }`)
- Export `getDocks()` — query all docks ordered by name (for filter dropdown and form selects)
