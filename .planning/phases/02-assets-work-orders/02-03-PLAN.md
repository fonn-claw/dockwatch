**addPart(workOrderId: number, formData: FormData):**
- requireRole(["manager", "crew"])
- Zod schema: name (string 1-255), quantity (number int positive, default 1), unitCost (number, in dollars — multiply by 100 for cents storage), notes (optional string)
- Verify work order exists and is not deleted
- Insert into workOrderParts: { workOrderId, name, quantity, unitCost: Math.round(parsed.unitCost * 100), notes }
- Call logAudit: action="add_part", entityType="work_order", entityId=workOrderId, metadata={ partName, quantity, unitCost }
- revalidatePath(`/work-orders/${workOrderId}`)
- Return { success: true }

**removePart(partId: number, workOrderId: number):**
- requireRole(["manager", "crew"])
- Delete from workOrderParts where id=partId
- Call logAudit: action="remove_part", entityType="work_order", entityId=workOrderId, metadata={ partId }
- revalidatePath(`/work-orders/${workOrderId}`)
- Return { success: true }

**updateTimeSpent(workOrderId: number, minutes: number):**
- requireRole(["manager", "crew"])
- Validate minutes is a positive integer
- Update workOrders set timeSpentMinutes=minutes, updatedAt=new Date() where id=workOrderId
- Call logAudit: action="update_time", entityType="work_order", entityId=workOrderId, metadata={ minutes }
- revalidatePath(`/work-orders/${workOrderId}`)
- Return { success: true }

**updateWorkOrderNotes(workOrderId: number, notes: string):**
- requireRole(["manager", "crew"])
- Update workOrders set notes=notes, updatedAt=new Date() where id=workOrderId
- Call logAudit: action="update_notes", entityType="work_order", entityId=workOrderId
- revalidatePath(`/work-orders/${workOrderId}`)
- Return { success: true }
