**Create `src/lib/queries/costs.ts`:**

All cost queries accept a period parameter: "month" | "quarter" | "year". Calculate periodStart/periodEnd:
- month: startOfMonth(now) to endOfMonth(now)
- quarter: startOfQuarter(now) to endOfQuarter(now) (use date-fns startOfQuarter/endOfQuarter)
- year: startOfYear(now) to endOfYear(now)

All monetary values returned in CENTS. Display layer converts to dollars.

- `getCostSummary(period)` — Aggregate across all work orders in period (by workOrders.createdAt within range):
  - totalPartsCents: SUM(workOrderParts.unitCost * workOrderParts.quantity) for parts on WOs in period
  - totalLaborCents: SUM(workOrders.timeSpentMinutes) * LABOR_RATE_PER_MINUTE for WOs in period
  - totalCostCents: totalPartsCents + totalLaborCents
  - workOrderCount: count of WOs in period
  - Return `{ totalPartsCents, totalLaborCents, totalCostCents, workOrderCount }`

- `getCostByDock(period)` — Group costs by dock:
  - Join workOrders -> docks for dock name
  - For each dock: partsCents (SUM parts), laborCents (SUM timeSpentMinutes * LABOR_RATE_PER_MINUTE), totalCents, woCount
  - Return array of `{ dockId, dockName, partsCents, laborCents, totalCents, woCount }` sorted by totalCents desc

- `getCostByCategory(period)` — Group costs by category:
  - Join workOrders -> assets for assetType
  - Map assetType to category using ASSET_TYPE_TO_CATEGORY
  - For each category: partsCents, laborCents, totalCents, woCount
  - Return array of `{ category, partsCents, laborCents, totalCents, woCount }` sorted by totalCents desc

- `getBudgetComparison(period)` — For each category in CATEGORY_BUDGETS:
  - Get actual spend from getCostByCategory (or inline the query)
  - Scale budget to period: month=budget/12, quarter=budget/4, year=budget
  - Budget values are in dollars in the config, convert to cents for comparison (multiply by 100)
  - Return array of `{ category, budgetCents, actualCents, percentUsed, isOverBudget }` sorted by percentUsed desc

- `getHighCostAssets(period)` — Per-asset cost totals:
  - Join workOrders -> assets, aggregate parts cost + labor estimate per asset
  - Return array of `{ assetId, assetName, assetType, dockName, partsCents, laborCents, totalCents, woCount }` sorted by totalCents desc, limit 20
